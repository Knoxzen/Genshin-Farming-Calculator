How to Use Our Website

  1. Use the Tab over to switch between character level up materials, weapon ascension materials, talent level up books or elemental stones.(Use Genshin Calculator to know how much material you need)
  2. Enter the quantities you have in you Backpack in the input box. Watch out for the background color of the materials you're entering.
  3. Press Calculate, and it will show all the possible conversions you can make.
  4. The results will be displayed in rows, indicating the conversions from one input to another. Row one shows the conversion from input 1 -> 2, row two shows the conversion from input 1 + 2 -> 3, and row three shows the conversion from inputs 1 + 2 + 3 -> 4.
